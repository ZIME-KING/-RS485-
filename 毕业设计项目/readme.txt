1、熟悉数字电路、模拟电路的基本应用电路；
2、掌握电路功能分析方法，通过软件模拟电路功能，搭建实物电路进行验证分析；
3、掌握实物试验数据分析工具以及分析方法；
4、掌握RS485元器件的特性和使用方法；
5、掌握电压和电流检测方法以及实现条件。
本课题是进一步地巩固前期课程，结合模拟信号检测技术和通信技术，设计一个具有通信功能的电池电压、电流监测系统。
主要内容：
1）、制作硬件测试平台；
2）、完成PCB设计工作；
3)、RS485通信距离3米，通信速率2Mbps；
4）、电压监测精度：土0.1V（0~30V），电流监测精度：土0.1A（0~5A）；
5）、本地和远程监控端实时显示电池电压电流等电参数；
6）、至少需要获取3分钟电池放电状态的电压电流等电参数数据，并绘制曲线，分析与其它仪器获得数据误差分析。
本课题进一步巩固学生基础电路应用、微控制器开发与应用、温度元件特性、隔离元件以及PCB制作应用。学生必须具备相关的硬件基础、软件使用能力、基本的自动控制知识；较强的动手实践能力，同时较强的责任心、耐心也必不可少。
